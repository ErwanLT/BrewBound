# Changelog

## 1.0.0

Premiere version stable de BrewBound.

- Carte interactive des brasseries artisanales.
- Catalogue des bieres par brasserie.
- Contributions collaboratives via Pull Requests GitHub.
- Validation serveur des soumissions avec Zod.
- Protection de l'API de contribution avec rate limit.
- Geocodage automatique des adresses de brasseries.
- Support Markdown pour les descriptions.
- CI GitHub avec typecheck, tests unitaires et build.
- Deploiement public sur Render.
